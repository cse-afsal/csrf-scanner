# CSRF Vulnerability Scanner

## About
This project demonstrates:
- Research on CSRF (Cross-Site Request Forgery)
- A Python crawler to detect missing CSRF tokens
- A CSRF Proof of Concept (PoC) attack page

## Tools Used
- Python
- Requests
- BeautifulSoup

## Disclaimer
This project is for educational purposes only.
Testing was done on vulnerable test applications like DVWA.
